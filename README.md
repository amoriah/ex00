Базовая работа с React + TypeScript + Style-components.

## Start

```bash
git clone https://github.com/amoriah/ex00.git
cd ex00
npm i
npm start
```
В приложении 3 задания: на изменение стилей по клику, на работу с датой и временем, на работу с формой.
